# Инструкция по использованию терминала в операционной системе nlix.
# 20.04.2018

Содержание:
1. Получение доступа к терминалу.
2. Использование sh
3. Список программ для терминала.

1. Получение доступа к терминалу.
Чтобы получить доступ к терминалу nlix, нужно подключить устройства ввода и вывода.
В качестве устройств можно использовать:
	EGP Screen
	Text Screen
	Console Screen

Примечания:

1. При использовании EGP Screen может запуститься графический режим вместо терминала. В этом случае нужно либо выйти из графического режима в режим терминала (menu -> exit -> sh) либо запустить виртуальный терминал (menu -> base -> xterm). При запуске программы в оконном менеджере xwm вы можете использовать только Keyboard для ввода, а в режиме терминала ещё и Text Entry.

2. Перед установкой Text Screen необходимо выставить минимальный размер текста и моноширинный шрифт, например, Courier New. В противном случае перенос текста будет работать некорректно.

В качестве устройства ввода можно использовать:
	Keyboard
	Text Entry

После подключения этих устройств вы получите терминал.

2. Использование sh
По умолчанию в сессии терминала запускается коммандный интерпретатор sh. Чтобы запустить программу, нужно написать её название на клавиатуре и нажать Enter.

Программу можно запустить с аргументами. Это может выглядеть так:

sh $ print-pi 1000

Где print-pi - имя программы, а 1000 - аргумент.

Также sh позволяет запустить несколько программ так, чтобы выходной поток одной программы являлся входным для другой. Например:

sh $ echo 2+2 > calc

Тогда echo отправит аргумент "2+2" во входной поток калькулятора calc, а он посчитает это выражение и выведет "4".

3. Список программ для терминала.
auto-ws - создаёт новое рабочее место.
dlist - выводит список устройств.
devcmd - позволяет управлять девайсами в ручном режиме, то есть, менять значения их входов и просматривать значения их выходов.
dremove [id/type] - удаляет девайс с указанным id или все девайсы указанного типа.
wc [command] - позволяет подключить внешнее устройство, не используя wire tool. Требует дополнение wiring для работы.

echo [args] - выводит все введённые аргументы.
help - выводит информацию о nlix'е и список программ.
rund [program name] - позволяет запустить приложение на фоне. Программа, запущенное через rund, не закроется после закрытия терминала.

plist - возвращает список исполняемых на данный момент программ.
kill [program id] - завершает выполнение программы с указанным id.
reboot - перезапускает систему.
poweroff - выключает систему и останавливает работу чипа.

perf [number 0-100] - устанавливает ограничение максимальной нагрузки. Не работает при запущенном autoperf. Выполнение без аргументов выводит текущее ограничение максимальной нагрузки.
overload [number] - устанавливает степерь перегрузки. Чем выше перегрузка, тем быстрее работают программы, но тем выше шанс падения системы с ошибкой tick quota.
uptime - выводит время работы системы.
print-pi [max iterations] - тест производительности. Вычисляет сумму ряда, стремящегося к числу pi и выводит потраченое на вычисление время.

ls [path] - выводит содержимое указанной директории.
cat [path] - выводит содержимое указанногго файла.
touch [path] - создаёт пустой файл по указанному пути. При чём `echo 2+2 > touch /rdir/testfile.txt` создаст файл с именем "testfile.txt" в директории "/rdir/" с содержимым "2+2".
rm [path] - удаляет указанный файл.
cp [path from] [path to] - копирует файл.

calc - консольный калькулятор. Поддерживает бинарные операторы + - * / ^, а также скобки любой вложенности.
surl [url/command] - плеер музыки. Только для серверов с дополнением E2Power.
digi-scan [res x] [res y] - сканнер местности для Digital Screen.
wget [url] [path] - программа для скачивания веб-страницы. Если не указан путь, сохраняет в файл webpage.
ssh - удалённое управление другим чипом nlix.
programmer [path] [name] - заливает в подключенный чип содержимое указанного файла. Если не указывать имя чипа, будет заливать в чип с имененм generic.

xstart - запускает программу для графической системы x на отдельном EGP экране.
